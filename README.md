# Challenger Soluevo - YC

## Descrição
Simulação de catalogo de produto! Desafio YC - Soluevo, por Fabio Vinteira

## Como rodar a aplicação
- Instalar dependências
```sh
$ npm install
```

- Buildar aplicação
```sh
$ npm run build
```

- Rodar aplicação no modo de desenvolvimento
```sh
$ npm run dev
```

- Executando via browser  
http://localhost:3000
